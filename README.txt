Если зайти под пользователем который забанен, то доступа к сайту больше не будет.

Для того чтобы получить доступ к сайту забанненому пользователю, можно убрать эти строки в profile.php

if (!$_SESSION['user']['banned']) {
        header('Location: error.html');
    }
